## Assignment5_Local Storage

## Overview
* This project is a .Net maui application that allows users to create, edit and save their profile information using local storage.
  
## Features
## Input fields:
* Name
* Surname
* Email
* Bio

  * Save data in JSON format.
  * Store data in a local text file.
  * Automatically load and display saved data on on app start.

  ## How it works
  * User enters profile details.
  * Clicks Save - data is stored as JSON.
  * When app reopens - saved data is automatically loaded.
